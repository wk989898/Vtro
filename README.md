# Vtro

## 介绍

trojan 的 windows 图形界面

## 注意

- 可以订阅和添加节点
- 暂时无法更改节点选项,可以删除后再添加
- 支持 ping,tcp-ping

## 运行

```
npm run dist
cd dist
```
